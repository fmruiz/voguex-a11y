# **useReadImagesAlt**

This hook allows to read using speech synthesis the alternative text of the images.

## **Resources** :

- [Accessibility: Image Alt text best practices](https://help.siteimprove.com/support/solutions/articles/80000863904-accessibility-image-alt-text-best-practices)
- [Colorado State University: Alternative Text](https://www.chhs.colostate.edu/accessibility/best-practices-how-tos/alternative-text/)

## **Example** :

```tsx
    const { setReadAltImages } = useReadImagesAlt();

    return (
        <section>
            <button
                aria-label="button to read the alternative text of the images"
                onClick={() => setReadAltImages(true)}
            >
                Read alternative text of the images
            </button>
            <img src="image1.jpg" alt="Image one - Alternative Text" />
            <img src="image2.jpg" alt="Image second - Alternative Text" />
            <img src="image3.jpg" alt="Image third - Alternative Text" />
        </section>
    );
```
